A redox equation can be balanced using the following stepwise procedure: (1) Divide the equation into two half-reactions. (2) Balance each half-reaction for mass and charge. (3) Equalize the number of electrons transferred in each half-reaction. (4) Add the half-reactions together. In this video, we'll use these steps to balance the redox equation for the reaction between aluminum metal and hydrogen ions.

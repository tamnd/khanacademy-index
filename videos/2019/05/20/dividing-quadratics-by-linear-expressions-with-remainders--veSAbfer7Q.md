## Description

The result of dividing (x²+5x+8) by (x+2) is (x+3) with a remainder of 2. This means there is no polynomial we can multiply by (x+2) to get (x²+5x+8). Instead, we need to multiply (x+2) by (x+3)+2/(x+2).

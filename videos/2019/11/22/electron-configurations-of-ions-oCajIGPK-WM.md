To find the electron configuration for an ion, first identify the configuration for the neutral atom. Then, add or remove electrons depending on the ion's charge. For example, to find the configuration for the lithium ion (Li⁺), start with neutral lithium (1s²2s¹). Then, since the lithium ion has one less electron, remove an electron from the 2s subshell to get 1s².

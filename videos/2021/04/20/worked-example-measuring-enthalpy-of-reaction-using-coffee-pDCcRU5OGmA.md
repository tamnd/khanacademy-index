## Description

Constant-pressure calorimetry is often used to determine the changes in enthalpy for aqueous reactions. In this video, we'll see how data from a coffee-cup calorimetry experiment can be used to calculate ΔH_rxn for the reaction between AgNO₃(aq) and NaCl(aq).

## Description

Hess's law states that if a process can be expressed as the sum of two or more steps, the enthalpy change for the overall process is the sum of the ΔH values for each step. To use Hess's law, two principles must be understood: one, if an equation is reversed, the sign of the ΔH value is also reversed. Two, if an equation is multiplied by a coefficient, the ΔH value is multiplied by the same coefficient.

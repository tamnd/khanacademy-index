## Description

We can determine whether a salt solution will be acidic, basic, or neutral by considering the reactivity of both the cation and the anion with water. If neither species reacts with water, the solution will be neutral. If only the cation reacts with water, the solution will be acidic. If only the anion reacts with water, the solution will be basic. If both species react with water, the pH of the solution will depend on the relative strengths of the cation and the anion.

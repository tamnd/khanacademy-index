## Description

The solubility product constant, Kₛₚ, is an equilibrium constant that reflects the extent to which an ionic compound dissolves in water. For compounds that dissolve to produce the same number of ions, we can directly compare their Kₛₚ values to determine their relative solubilities. If we know the solubility of a salt, we can use this information to calculate the Kₛₚ value for the compound.

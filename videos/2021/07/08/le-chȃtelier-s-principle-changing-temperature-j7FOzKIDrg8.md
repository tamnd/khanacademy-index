Le Chȃtelier’s principle can be used to predict the effect that a stress like changing temperature has on a system at equilibrium. If the temperature of the system is increased (at constant V), the system will shift in the direction that consumes the excess heat. If the temperature of the system is decreased, the reverse effect will be observed.
